# Button 元件使用

## 引入

```javascript
import { Button } from "~~components";
```

## 屬性

- `color`: yellow、white、red、blue、black、transparent、transparentInModal、green、auto
- `disabled`
- `onClick`: event function

```html
<button color="yellow">View Detail</button>
```
